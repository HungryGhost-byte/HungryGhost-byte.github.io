Coding Project

Cards of Alteria

One of my personal projects is a game writtin in Python and Pygame.
The game is both a choose-your-own-adventure style game in which you get to decide where to go and what to do and a card collection game. Earn gold in the text adventure to spend in the shop to buy cards.
All code was by me, in which I had to figure out all sorts of things such as how to handle duplicate cards, how to show earned cards in an album using various data structures, class methods to create the cards themselves, and more.
You can find a lot of my coding work on my github with the top link or the one just below.
